# src

アプリケーションのソースコードを管理するディレクトリです。

## 役割
- サーバーの設定と構築
- APIルートハンドラーの実装
- ビジネスロジックの実装
- データベース操作の実装

## 含まれるディレクトリ
- `clients/` - 外部APIクライアントの実装
- `configs/` - アプリケーション設定の管理
- `databases/` - データベース関連の実装
- `middlewares/` - ミドルウェアの実装
- `models/` - データモデルの定義
- `repositories/` - データアクセス層の実装
- `routers/` - APIルートの定義
- `schemes/` - リクエスト/レスポンススキーマの定義
- `seeders/` - テストデータの定義
- `usecases/` - ビジネスロジックの実装
